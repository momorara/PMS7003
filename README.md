# PMS7003
PMS7003による空気質センサーのプログラムです。<br>

<<概要>>
　花粉症にお悩みの方は、花粉時期や黄砂、ハウスダストの状態が気になりますよね。<br>
 私も杉と檜に反応しますし、家で片付け物をしているとハウスダストに反応してしまいます。<br>
 そこで、ラズパイとPMS7003による空気質センサーをサクってみました。<br>
 花粉、黄砂、ハウスダクトを区別するあるごりずむもあるようですが、とにかく何か飛んでいるかを関知するため<br>
 PMS7003でPM1.0、PM2.5、PM10と合計値をambientに送信して可視化することとしました。<br>
 合計値には科学的な意味はないですが、感覚的には良い感じです。<br>
 全てpythonで作成していますので、ご自身の感覚に合わせて調整して下さい。<br>
 
すべてのソースプログラムを開示いたします。

・LEDの色等指定はできません。
・部品の仕様、有無等変わる場合があります。
・基板のバージョンが変わる場合がありますが、機能等に違いはありません。
・回路に関しても随時変更しますが、機能等に違いはありません。
・ラズパイは付属しません。
